
# Mateus Workbench

Repositório criado para estruturar automações, análises e materiais de apoio ao trabalho de implantação em investimentos correntes.

## Objetivo
Este repositório existe para apoiar rotinas ligadas a:
- aderência de projetos
- aderência física
- aderência financeira
- programação semanal aderente
- consolidação de indicadores
- geração de resumos executivos
- apoio à comunicação com liderança
- estruturação de planos de ação

## Contexto de uso
O repositório foi pensado para apoiar um fluxo de trabalho com foco em:
- gestão de obras
- controle e aderência
- análise de indicadores
- automação de tarefas repetitivas
- geração de saídas executivas
- padronização de entregas

## Estrutura inicial
A estrutura deste repositório será expandida para incluir:
- instruções para GitHub Copilot
- agentes customizados
- templates
- documentação de contexto
- automações em Python
