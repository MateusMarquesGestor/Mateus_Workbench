
# Template - Plano de Ação Semanal

## Objetivo
Este modelo serve para organizar ações semanais relacionadas a:
- aderência de projetos
- aderência física
- aderência financeira
- programação semanal
- segurança
- tratativas com liderança
- recuperação de desvios

---

## Estrutura resumida
Use este modelo para registrar de forma prática:
- qual é o problema
- qual é a causa provável
- qual ação será feita
- quem responde
- quando deve acontecer
- qual a prioridade
- qual o status da tratativa

## Modelo em formato de lista prática

### Ação
- o que será feito

### Responsável
- quem vai conduzir

### Prazo
- até quando

### Prioridade
- alta / média / baixa

### Status
- não iniciado / em andamento / concluído / bloqueado

### Critério de conclusão
- como saber que a ação foi concluída

### Observação
- registro complementar

---

## Orientações para uso
Ao preencher este template:
- usar linguagem clara
- evitar descrições genéricas
- registrar ações objetivas
- vincular ação ao desvio observado
- priorizar o que gera mais impacto
- atualizar o status com frequência
- usar este material como base para alinhamento e comunicação
